# 🤖 DecodeBot - Rule-Based AI Chatbot

## Overview

DecodeBot is a simple rule-based chatbot developed in Python. The chatbot interacts with users by recognizing predefined keywords and responding with appropriate messages. This project demonstrates the basic concepts of Artificial Intelligence, conditional logic, and conversational systems.

The chatbot continuously accepts user input and generates responses based on a set of predefined rules.

## Features

* Interactive command-line chatbot
* Responds to greetings such as "hello" and "hi"
* Answers simple questions
* Provides help information
* Handles unknown queries gracefully
* Allows users to exit the conversation using the "bye" command

## Technologies Used

* Python 3
* Conditional Statements (`if-elif-else`)
* Loops (`while`)

## Working Principle

The chatbot uses a rule-based approach where each user input is compared against predefined conditions. If a match is found, the corresponding response is displayed. Otherwise, the chatbot informs the user that it does not understand the query.

## Sample Conversation

```text
You: hello
Bot: Hi there!

You: name
Bot: My name is DecodeBot.

You: how are you
Bot: I'm just a bot, but I'm doing great!

You: bye
Bot: Goodbye!
```
